If you want to add a location you can fist check the extended database (with Add-from-database) to see whether you can add it from there.

If the location is not available in the extended database you can create a new location:

  • A real geographic location can be added to the database with 'add geo location'. If a city location is added it is important to add latitude and longitude information (can be found on wikipedia or google maps). Once the geolocation is created you should use Add-from-database to add it to the available locations.
  • A fictional location or a combination of multiple (geographic) locations can be added with add user location.
  • You can edit locations by searching them in the geoloc (geolocations) or userloc (user location) list and clicking the edit button.
